Types of Numbers


Natural Numbers : Counting numbers are called natural numbers.
Thus 1, 2, 3, 4, .... are all natural numbers.


Whole Numbers : All counting numbers or natural numbers and 0(zero) form the set of whole number.
Thus 0, 1, 2, 4, .... are all whole numbers.


Integers Numbers : All counting numbers, zero and negative of counting numbers form the set of integers.
Thus ... -2, -1, 0, 1, 2, ... are all intergers.

Rational Numbers : Any number that can be expressed in the form p upon q (p/q) and q≠0 is called a rational number.
Example :
1/3 = 0.33
-3/-2 = 1.5


Irrational Numbers : Any number that can not be expressed in the form p upon q (p/q) is called a irrational number.
Example :
π = 22/2 = 3.141592653...
e = 2.7182818...
√(2) = 1.414213...


Real Numbers : Real numbers are numbers that can be found on the number line. This includes both the rational and irrational numbers.


Example:
Set of positive integers = (1,2,3,4,5,6, ...).
Set of negative integers = (-1,-2,-3,-4,-5,-6, ...).
Set of all non-negative integers = (0,1,2,3,4, ...).
